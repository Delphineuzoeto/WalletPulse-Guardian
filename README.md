# WalletPulse Guardian

Explainable pre-sign wallet risk scanner for Solana. Built for the Dev3pack Global Hackathon.

## Stack

- Python (standard library + solders for transaction parsing)
- HTML / CSS / vanilla JavaScript
- Helius (live Solana wallet history)
- OpenAI (verdict explanation)
- ElevenLabs (voice warnings)
- Solana Actions / Blinks (shareable verdict cards)
